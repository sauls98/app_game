# Videos locales

Coloca aquí videos propios si quieres ampliar la experiencia offline, por ejemplo:

- `ambient.mp4`
- `intro.mp4`
- `reward-loop.mp4`

Para usarlos, agrega una etiqueta `<video>` en `index.html` o carga el archivo desde `app.js` con una ruta local como `assets/video/ambient.mp4`.
No uses enlaces externos si quieres mantener la app completamente offline.
